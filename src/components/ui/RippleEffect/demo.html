🎨 Custom Styled Button

Note: The ripple effect works with any button style! Just add the 'ripple-container' class.

📝 How to Implement

1. Add the CSS class to your buttons:

<button class="ripple-container btn-primary">Add to Cart</button> <button class="ripple-container btn-secondary">Make an Offer</button> <button class="ripple-container btn-ghost">Contact Seller</button>

2. Include the CSS and JavaScript files:

<link rel="stylesheet" href="RippleEffect.css"> <script src="RippleEffect.js"></script>

3. For React components:

import RippleButton from './components/ui/RippleEffect'; <RippleButton variant="primary" onClick={handleAddToCart}> Add to Cart </RippleButton>

Customization: Change the ripple color by modifying the 'background-color' property in the '.ripple' CSS class. The default is a soft purple: rgba(147, 51, 234, 0.3)
